체크박스 리스트 egovCheckboxList

체크박스 리스트를 보다 쉽게 사용하기 위해 개발되었습니다.

Usage

체크박스 리스트 컴포넌트는 기본적으로 아래와 같이 태그를 입력하여 적용합니다.




				


			
parameter

			
체크박스 리스트 컴포넌트의 주요 파라메터는 아래와 같습니다.

			

				
egovCheckboxList(array)

				
체크박스 리스트를 구성할 리스트 값입니다.


				
ngModel(array)

				
체크박스 리스트의 선택된 값 입니다. 체크박스 리스트의 모델값은 여러 선택된 정보를 가지기 때문에 Array 형의 데이터를 사용합니다. 기본 값은  [ ] 입니다.

				
				
egovCheckboxName(string) - optional

				
체크박스 리스트의 옵션으로 실제 보여질 값의 필드명 입니다. 기본값은 "name" 입니다.


				
egovCheckboxValue(string) - optional

				
체크박스 리스트의 옵션으로 실제 값의 필드명 입니다. 기본값은 "value" 입니다.


				
egovCheckboxDirection(string) - optional

				
체크박스 리스트의 옵션으로 체크박스 구성시 구성 방향입니다. 수평 방향으로 진행 시 "h", 수직 방향으로 진행시 "v" 값을 사용합니다. 기본값은 "h" 입니다.


				
egovCheckboxSelectall(string) - optional

				
체크박스 리스트의 옵션으로 "전체" 값입니다. 리스트에 "전체"에 해당되는 값이 존제 하는 경우 다음과 같은 동작을 합니다. "전체" 값에 해당되는 체크박스가 선택되면 기존의 선택된 모든 체크를 해지하고 "전체" 값만 선택이 됩니다. 반대로 "전체" 값이 선택이 되어있는데 다른 값을 선택하면 "전체" 값은 체크 해지가 됩니다. 기본 값은 "" 입니다.


				
egovCheckboxDisabled(boolean) - optional

				
체크박스 리스트의 옵션으로 체크박스 리스트의 비활성 여부입니다. 기본값은 false 입니다.

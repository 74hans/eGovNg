셀렉트 리스트 egovSelectList

셀렉트 리스트를 보다 쉽게 사용하기 위해 개발되었습니다.

Usage

셀렉트 리스트 컴포넌트는 기본적으로 아래와 같이 태그를 입력하여 적용합니다.




				


			
parameter

			
셀렉트 리스트 컴포넌트의 주요 파라메터는 아래와 같습니다.

			

				
egovSelectList(array)

				
셀렉트 리스트를 구성할 리스트 값입니다.


				
ngModel(number)

				
셀렉트 리스트의 선택된 값 입니다.

				
				
egovSelectName(string) - optional

				
셀렉트 리스트의 옵션으로 실제 보여질 값의 필드명 입니다. 기본값은 name 입니다.


				
egovSelectValue(number) - optional

				
셀렉트 리스트의 옵션으로 실제 값의 필드명 입니다. 기본값은 value 입니다.

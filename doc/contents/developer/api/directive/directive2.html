차트 Chart

정보의 시각화 도구로 차트는 많이 사용 되고 있습니다. 주로 사용되는 차트들을 사용하기 쉽게 개발하였습니다.

Usage

차트 컴포넌트는 기본적으로 아래와 같이 태그를 입력하여 적용합니다.




				

			
사용할 수 있는 차트의 종류는 egovLineChart, egovCumulativeLineChart, egovStackedAreaChart, egovMultiBarChart, egovDiscreteBarChart, egovHistoricalBarChart, egovMultiBarHorizontalChart, egovPieChart, egovScatterChart, egovLinePlusBarChart, egovLineWithFocusChart, egovBulletChart, egovSparkline 등이 있습니다. 
			



			
parameter

			
차트 컴포넌트의 주요 파라메터는 아래와 같습니다.

			

				
id(string)

				
차트의 실별자 입니다. 차트를 만들고 업데이트 할 때 사용되며, 한페이지에 여러 차트가 존재할 경우 모든 차트는 고유 ID가 있어야 합니다.

				
				
data(string)

				
Angular 표현식의 형태로 데이터 바인딩(data-bind) 합니다. 차트로 표현될 정보는 다음과 같습니다. [{"key" : "데이터 명", "values" : [[x축 data, y축 data], [x축 data, y축 data],[x축 data, y축 data], ... ]}


				
width(number)

				
차트의 넓이입니다.


				
height(number)

				
차트의 높이입니다.


				
margin(object)

				
차트 내부의 여백 크기 입니다. 실제 차트 그래프 외의 정보(축정보, 레전드 등)의 공간 크기를 결정합니다. 다음과 같이 값을 지정할 수 있습니다. "{left:50,top:20,bottom:50,right:50}"
